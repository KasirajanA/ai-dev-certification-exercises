# AI-Driven Developer Certification — Exercise Starter Code

Starter projects for the **Certified AI-Driven Developer** certification program.

## Quick Start

```bash
# Fork this repo, then clone your fork
gh repo fork a6unn/ai-dev-certification-exercises --clone

# Navigate to any exercise folder and start Claude Code
cd module-1-taskmaster
claude
```

## Exercises

| Folder | Module | Exercise | Description |
|--------|--------|----------|-------------|
| `module-0-weather-api/` | 0: Zero to Running | Weather Dashboard API | Scaffold for building your first AI-generated project |
| `module-1-taskmaster/` | 1: Your First AI-Powered Feature | Add Priority (CRISP) | Pre-built task manager — practice CRISP prompting |
| `module-1-commitgen/` | 1: Your First AI-Powered Feature | Git Commit Message Generator | Scaffold for building a real developer tool |
| `module-2-debugger/` | 2: Navigate, Debug & Understand | Debug Detective | Express API with 3 intentional bugs to find and fix |
| `module-3-url-shortener/` | 3: Build with Confidence | URL Shortener TDD | Practice Red-Green-Refactor with Vitest |

## How to Use

1. **Fork** this repository to your GitHub account
2. **Clone** your fork locally
3. **Navigate** to the exercise folder for your current module
4. **Launch** Claude Code with `claude` and follow the exercise instructions in the certification platform
5. **Commit** your work as you go — your git history is part of the learning

## Requirements

- [Claude Code](https://code.claude.com/docs/en/quickstart) installed
- Node.js 18+ (for npm-based projects)
- Git
