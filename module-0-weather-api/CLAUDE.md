# Weather API Project

## Tech Stack
- Node.js + TypeScript + Express
- No external API keys needed — use mock weather data

## Code Style
- Use async/await (no callbacks)
- Functional style, no classes
- 2-space indentation

## Project Goal
Build a weather dashboard API with:
1. GET /health — returns status and uptime
2. GET /weather/:city — returns mock weather data
3. GET /forecast/:city — returns 5-day mock forecast
4. GET / — HTML dashboard showing weather for 5 cities
